Color Schemes

If you would like to change color scheme, find stylesheet with the id of theme-color in index.html file and replace href attribute with the following color options.

default, pink, teal, green, light-green, amber, orange

Social Links

If you would like to put your social links, find Social Icons label within home section in index.html file and add links in the href attribute with your respective links.

Background Image

If you would like to change the background image, add the following code before </header> tag in index.html file.

<style>
main {
    background-image: url(images/backgound.jpg) !important;
}
</style>

Replace backgound.jpg with your image name.

Name & Skills

If you would like to edit the name and skills, find Name & Skills label within home section in index.html file and replace the information with yours.

The animated skills are in a div with js-rotating id, you can add as many as can with comma separated values.

If you would like to change the animation effect of skills, findAnimated Skills section in main.js file and replace fadeInDown with your desired effect.

You can see the complete list of animation effects here https://daneden.github.io/animate.css/.

Animations

This theme has a function for animations which depends upon animate.css.

To apply animation to any element you just need to add the data-anim-effect attribute with the value of animation name.

If you want to delay an animation for an element you can add data-anim-delayattribute with the value in milliseconds. For reference 1000milliseconds equals to 1 second.

You can see the complete list of animation effects here https://daneden.github.io/animate.css/.

About Section

If you would like to edit this section, simply find the About Section label in the index.html file, and then scroll down until you find the appropriate area that needs to be edited.

Resume Section

The filters you see at the top of this section are wrapped in a div with id filter-resume-group. All the <a> tags has data-group attribute which links to the class name it filters.

If you would like to add a timeline block, copy the div with class name timeline-block and paste it. To remove a block simply delete the div with class name timeline-block.

If you would like to add a chart, you would do the following:

Bar Chart

For Bar Chart, add a canvas tag with a unique id and data-chart-type="bar" attribute like the following:

<canvas id="front_end_skills" data-chart-type="bar" data-labels='["HTML5", "CSS3", "SASS", "jQuery", "JavaScript", "AngularJS"]' data-values="[80, 80, 80, 80, 80, 60]" data-colors='["#FF6384", "#4BC0C0", "#FFCE56", "#4BC0C0", "#36A2EB", "#FF6384", "#E7E9ED"]' data-heading="Front End Skills"></canvas>
Doughnut Chart

For Doughnut Chart, add a canvas tag with a unique id and data-chart-type="doughnut" attribute like the following:

<canvas id="back_end_skills" data-chart-type="doughnut" data-labels='["PHP", "SQL", "WordPress"]' data-values="[20, 20, 60]" data-colors='["#FF6384", "#36A2EB", "#FFCE56"]'></canvas>
Pie Chart

For Pie Chart, add a canvas tag with a unique id and data-chart-type="pie" attribute like the following:

<canvas id="web_tools" data-chart-type="pie" data-labels='["Git", "GruntJS"]' data-values="[50, 50]" data-colors='["#FF6384", "#FFCE56"]'></canvas>

In all the above chart examples: data-labels, data-values, data-colors are required attributes and should be in a valid JSON format and should contain the same number of values in all three attributes. data-heading attribute is also required in bar chart.

Progress Bar

If you would like to add a progress bar, you would do the following:

<div class="ui red progress" data-percent="80">
    <div class="bar">
        <div class="label">Adobe Flash</div>
        <div class="progress"></div>
    </div>
</div>

To change the percentage, change the value in data-percent attribute. To change the color of progress bar, you can replace the class name red with the following color options.

red, orange, yellow, olive, green, teal, blue, violet, purple, pink, brown, grey, black

Rating

If you would like to add rating, you would do the following:

<div class="rating">
    <div class="content">
        <div class="header">
            Spanish
            <div class="ui small star rating" data-rating="4" data-max-rating="5"></div>
        </div>
    </div>
</div>

data-max-rating contains the maximum number of rating and data-rating contains how much do you rate.

Portfolio Section

The filters you see at the top of this section are wrapped in a div with id filter-portfolio-group. All the <a> tags has data-group attribute which links to the data-groups attribute value it filters.

A portfolio item consist of the following code:

<div class="portfolio_item column" data-groups="web">
    <div data-anim-effect="zoomIn" data-anim-delay="100">
        <a href="#popup-1" class="open-popup-link">
            <img src="images/picture.jpg" title="View Project" alt="" />
        </a>
    </div>
    <div id="popup-1" class="white-popup mfp-hide">
        <div class="ui items">
            <div class="item">
                <div class="ui image">
                    <img src="images/picture.jpg" alt="">
                </div>
                <div class="content">
                    <div class="header">Project Title</div>
                    <div class="meta">
                        <span>Date</span>
                        <span>Category</span>
                    </div>
                    <div class="description">
                    .......
                    </div>
                </div>
            </div>
        </div>
    </div>
</div>

data-groups attribute contains the filter value.

#popup-1 should be a unique id for every portfolio item.

All the portfolio items are wrapped in a div with portfolio_container id.

Contact Form

Mailgun is integrated in this theme for uninterrupted emails. You can use this free service for 10,000 emails every month. Visit mailgun.com for more information.

It is already integrated and ready to use. You just need to put your domain details & api-key in mailgun-settings.php file.

  1. Create account on mailgun.com for free.
  2. You can use sandbox domain that you get or you can add your own domain with some simple steps provided at their website.
  3. You will need two important factors from your mailgun account. 1. Domain Name. 2. API Key
  4. Click on 'domains' link from navigation.
  5. Select the sandbox domain.
  6. On this screen on mailgun.com, you will get both details.
  7. Open mailgun-settings.php file which is included in the theme's root folder.
  8. Replace the mailgun domain name in $mailgunDomain variable & API key in $mailgunKey variable.
  9. Replace the email address in $yourEmail variable with you own, where you want to receive emails from website.

Make sure you have uploaded 'Mailgun' folder provided with your theme download.
